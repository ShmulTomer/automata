How to Use the Automata Designer

  • Add a state: Double-click on the canvas.
  • Add an arrow: Shift-drag on the canvas.
  • Move something: Drag it around.
  • Delete something: Click it and press the delete key (not the backspace key; on Mac use fn + delete).
  • Make accept state: Double-click on an existing state.
  • Type numeric subscript: Put an underscore before the number (like "S_0").
  • Type Greek letter: Put a backslash before it (like "\beta").
  • Type special symbols: Put a backslash before it (like "\rightarrow", "\sqcup", or "\emptyset").
  • Backup/restore diagram: Download JSON here or upload here.
